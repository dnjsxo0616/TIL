# 2. semantics in HTML

## HTML semantic Element
- 기본적인 모양과 기능 이외에 의미를 가지는 HTML 요소
- 검색엔진 및 개발자가 웹 페이지의 콘텐츠를 이해하기 쉽게 만들어 준다.

## 페이지 구조화를 위한 대표적인 semantic element

- header
- nav
- main
- article
- section
- aside
- footer
