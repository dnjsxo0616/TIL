# 완전 탐색

## 브루트 포스 Brute-force(무식하게 다해보기)

- 가장 단순한 풀이 기법이며, 단순 조건문과 반복문을 이용해 풀 수 있다.

## 2. 델타 탐색(Delta Search)

- (0, 0)에서부터 이차원 리스트의 모든 원소를 순회하며(완전탐색)
- `각 지점에서 상하좌우에 위치한 다른 지점을 조회하거나 이동`하는 방식이다.

## 상하좌우

```python
x = 1
y = 1

delta = [(-1, 0), (1, 0), (0, -1), (0, 1)]
for dx, dy in delta:
    print(x+dx, y+dy)
```
```python
dx = [-1, 1, 0, 0]
dy = [0, 0, -1, 1]

for i range(4):
  nx = x + dx[i]
  ny = y + dy[i]
  # 범위를 벗어나지 않는지 확인하면서 갱신한다.
  if 0 <= nx <3 and 0< = ny < 3: # 3x3행렬
    x = nx
    y = ny
```
